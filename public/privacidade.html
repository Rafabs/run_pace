Política de Privacidade

Esta política descreve como coletamos, usamos e protegemos suas informações ao usar o site Run Pace.

1. Coleta de Informações

Não coletamos dados pessoais dos visitantes, exceto no login administrativo, onde são utilizados apenas para autenticação interna.

2. Cookies

Utilizamos cookies apenas para fins de funcionalidade e desempenho, não coletamos dados sensíveis ou comportamentais.

3. Compartilhamento de Dados

O Run Pace não compartilha dados de usuários com terceiros.

4. Segurança

Tomamos medidas para proteger eventuais dados fornecidos (como login), mas nenhuma transmissão pela internet é 100% segura.

5. Alterações

Podemos atualizar esta política periodicamente. Recomendamos que o usuário consulte esta página regularmente.
